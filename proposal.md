# Cs 30 major project

## Description

- A 2D Minecraft like game.

## Need To Have

- Trees for wood.
- Layers so player can mine (Dirt, grass and stone).
- Player can brake blocks and pick them up.
- Player has inventory.
- Map is random everytime.
- Player can place blocks and is able to craft tools with wood and stone.
- Mobs in game even if player can't interact with them.
- At least 5 block deep ground (like 2 blocks grass and dirt then 3 blocks of stone).

## Nice To Have

- Different ores like iron and diamond.
- Mobs can move around and player can get drops from them.
- Day and night cycle.
- Other dimensions like nether.
- Player can be in different modes like hardcore and creative.
- Player can change their skin.
- Boss fights like the Ender Dragon.
